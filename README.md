# 📊 Billionaires Statistics Dataset Analysis

This repository contains the **Billionaires Statistics Dataset (Raw)** used for data analysis and visualization projects. The dataset provides detailed information about billionaires worldwide, including their wealth, industry, demographic details, and country-level economic indicators.

---

## 📂 Dataset Features

- 🌍 Global billionaire rankings
- 💰 Net worth (Final Worth)
- 🏭 Industry and wealth source
- 🌎 Country and city information
- 📈 GDP and CPI by country
- 👥 Population statistics
- 💵 Tax revenue and total tax rate

---

## 🎯 Project Objectives

- Clean and preprocess raw data
- Perform Exploratory Data Analysis (EDA)
- Identify trends in billionaire wealth distribution
- Analyze industry-wise and country-wise billionaire concentration
- Create interactive dashboards using Microsoft Excel

---

## 🛠️ Tools & Technologies

- Microsoft Excel

---

## 📈 Possible Insights

- Top countries with the highest number of billionaires
- Distribution of wealth across industries
- Country-wise billionaire concentration
- Relationship between GDP and billionaire count
- Economic indicators influencing billionaire wealth
